# YouTube Transcript Hub (Pro Edition)

A high-performance, single-server Python application designed to concurrently extract, parse, and format transcripts for multiple YouTube videos. The application features a modern glassmorphic dashboard with dynamic mouse-tracking 3D perspective effects, millisecond timing precision, and real-time streaming Gemini AI integration.

## Key Features

* **Concurrent Batch Extraction**: Retrieves transcripts and metadata (titles, view counts, thumbnails) for multiple YouTube URLs simultaneously.
* **WebVTT Fallback Parsing**: Automatically falls back to downloading and parsing standard WebVTT/SRT subtitles if YouTube's high-accuracy JSON3 subtitles are unavailable.
* **Map-Reduce AI Summarization**: Employs a Map-Reduce execution model for long transcripts. The system summarizes chunks in parallel, consolidates the intermediate summaries, and generates a single cohesive summary, chapters list, or action-item outline.
* **AI Streaming Tasks**: Streams processed text chunk-by-chunk in real time using Server-Sent Events (SSE) and the Gemini API for tasks including beautification, summarization, action-item extraction, and chaptering.
* **Rich Export Facilities**: Exports transcripts to plain text (TXT), standard subtitles with millisecond accuracy (SRT or WebVTT), Markdown (MD), or formatted PDF documents.
* **Tactile 3D User Interface**: A glassmorphic dark-mode interface utilizing hardware-accelerated 3D tilt effects, glowing mesh canvas details, and interactive audio visualization waves. Includes a performance-optimized Lite Mode to disable all intensive animations and filters on lower-end devices.
* **XSS Defenses**: Full client-side HTML sanitization preventing Cross-Site Scripting (XSS) when rendering dynamic video metadata.

## Getting Started

### 1. Install Dependencies
Ensure you have the required Python libraries installed globally or in your virtual environment:
```bash
pip install yt-dlp google-generativeai
```

### 2. Launch the Application
Run the backend server:
```bash
python app.py
```
Or use the desktop launcher wrapper:
```bash
python youtube_transcript_app.py
```

### 3. Open in Browser
The launcher will automatically open your default browser, or you can navigate manually to:
```text
http://localhost:8000
```

## System Architecture

1. **Extraction**: The Python backend uses `yt-dlp` to extract metadata and retrieves the appropriate timed subtitle files.
2. **WebVTT Parsing**: If JSON3 is missing, the backend parses standard `.vtt` segments, strips inline styling tags, and filters out duplicate frames generated by YouTube's scrolling auto-captions while preserving millisecond timestamp details.
3. **Map-Reduce Processing**: For transcripts exceeding context limits, the backend splits the data, performs map summarizations concurrently, and executes a final reduction pass via the Gemini API, streaming the consolidated output using SSE.
