How to Use

YouTube Transcript Hub

Extract clean transcripts, metadata, views, and titles from multiple YouTube videos simultaneously. Perform streaming AI summarization, translation, and chaptering.
Provide your Gemini API Key to enable AI Streaming Tasks (Beautify, Summarize, Actions, and Chapters).
How to obtain a free API Key:
  1. Go to Google AI Studio.
  2. Log in with any standard Google account.
  3. Click the blue "Get API Key" button in the top left corner.
  4. Click "Create API Key", choose a project, and copy your generated key.
  5. Paste the key into the input field below to unlock AI tasks.
